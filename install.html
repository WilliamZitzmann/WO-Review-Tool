WO Review Tool

Install in one drag.

A bookmarklet that adds a live checklist panel to Maximo work order pages. No install, no extension — just a bookmark.

Drag the button below to your bookmarks bar
Review Tool
Clicking it here does nothing useful — it only works dragged to your bookmarks bar, then clicked from there while a Maximo work order is open.
Don't see your bookmarks bar? Show it first: Ctrl+Shift+B on Chrome/Edge, Ctrl+B on Firefox.

Then, to use it

1
Open a work order in Maximo.
2
Click the Review Tool bookmark you just added.
3
If it redirects you to the right Maximo page first, click the bookmark again once you've landed and are logged in.
4
First time only: the tool checks whether you have permission to use it, then opens an installer where you pick a starting configuration.
Can't drag it? Add it manually instead
1
Right-click your bookmarks bar → Add page (or create a new bookmark any way your browser allows).
2
Name it anything — e.g. Review Tool.
3
Paste the code below into the URL/address field instead of a real URL.
javascript:(function(){var LOADER_URL='https://wo-review-tool-access.williamzitzmann.workers.dev/loader.js';var CACHE_KEY='__wo_loader_cache';function run(src){try{eval(src);}catch(e){alert('WO Tool loader error: '+e.message);}}var xhr=new XMLHttpRequest();xhr.open('GET',LOADER_URL,true);xhr.onload=function(){if(xhr.status===200){localStorage.setItem(CACHE_KEY,xhr.responseText);run(xhr.responseText);}else{var c=localStorage.getItem(CACHE_KEY);if(c){run(c);}else{alert('WO Tool: failed to load (HTTP '+xhr.status+').');}}};xhr.onerror=function(){var c=localStorage.getItem(CACHE_KEY);if(c){run(c);}else{alert('WO Tool: network error, check connection.');}};xhr.send();})();
